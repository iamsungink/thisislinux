## 리눅스 기본 명령어 (1)
* ls
* cd
* pwd
* rm
* cp
* touch
* mv
* mkdir
* rmdir
* cat
* head, tail
    * head -5 file명 : 상위 5행까지 보여줌
    * tail -5 file명 : 마지막 5행만 화면에 출력
* more
    * 화면에 페이지 단위로 출력
* less
    * 화살표키나 pageup, pagedown 도 사용가능
* file  : 해당 파일이 어떤 종류의 파일인지를 표시해준다.
